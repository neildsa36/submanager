# 💳 SubManager

A full-stack Subscription Management System with email alerts and reports.

---

## 🚀 Features

- Add / Edit / Delete subscriptions
- Email reminders (due + expired)
- Active vs Expired tracking
- Monthly & yearly cost calculation
- PDF report generation
- Search + filter system

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MySQL
- HTML, CSS, JavaScript
- Nodemailer
- PDFKit

---

## ⚙️ Setup

1. Install Node.js  
2. Start MySQL (XAMPP)  
3. Import database (`submanager.sql`)  
4. Install dependencies:
   ```bash
   npm install
